Vindina Inc.

Vindina is a company that wants to create several applications for their consumers.

This is an improved version of a previous project, opting for more features and applications rather than having a simple manager setting.

All of the services was inspired from Google.

Tools used:

  • Bootstrap for the design to make it look professional.
  • PHP for the backend to validate inputs from users.
  • PHP connected to SQL to store the data entered by the user.
  • XAMPP for the deployment.
  • Second device that acts like the server.
  • Local internet for anyone connected can access to it.

Features:

  • Vindina: Mercury offers a bookmarks manager, like Google Save.
  • Vindina: Venus offers a presentation app, like Google Slides.
  • Vindina: Terra offers an AI chatbot to search the internet, like Google Search.
  • Vindina: Mars offers a document app, like Google Docs.
  • Vindina: Jupiter offers a free 5GB storage for files, like Google Drive.
  • Vindina: Saturn offers a spreadsheet app, like Google Sheets.
  • Vindina: Uranus offers a notes manager, like Google Keep.
  • Vindina: Neptune offers a password manager, like Google's Password Manager.
