Android Mobile App

Final project for ITS181-3 Application Development 3

This is a school project that focuses on creating a mobile app using Kotlin, Java for the backend, and MySQL for the database.

It simply gets the lists names from the database using Spring and displays it in the phone.

Tools used:

  • Android Studio for the design and the virtual phone.
  • Spring Boot for the server.
