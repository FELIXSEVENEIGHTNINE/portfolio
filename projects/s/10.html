Car Rental System

Final project for CSS123L Object-Oriented Programming

OFFGames is a small company that sells games. They have a website called OFFGame that sells offline games.

This project introduced me to an extensive use of Bootstrap, Data Management on SQL, and using APIs for the website and for the AI.

The website is hosted locally on my device while the AI is hosted locally on a group member's device. We used the same network to connect the AI to the website.

We, the developers, made a working gamified advertisement of Five Nights at Freddy's (cannot be sold for lawful reasons) with an achievement system and other minigames.

The project was used for two different courses.

Tools used:

  • Javascript
  • Bootstrap
  • PHP
  • Figma
  • XAMPP
  • Python
  • AI chatbot and recommendation system uses Ollama

Features:

  • Users can log in, register, and log out.
  • Users can view and edit their profile.
  • Users have a wallet which they can purchase games.
  • Users can use a chatbot to recommend games for them.
  • Users can also use a simple recommendation system that gives them 10 games from genres they chose from.

  • A hidden admin page for the admin to only view who are the users in the website.

  • An achievement system for the gamified advertisement.

  • Mobile compatable using any browser.
