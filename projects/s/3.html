VRMT

Final project for CSS152L Software Engineering 2

Fake is a fake company that sells electronics like monitors, PCs, motherboards, etc.

This project introduces me to e-commerce and what services they usually provide for users.

This project does not include a backend.

Tools used:

  • Bootstrap for the design.
  • Github Pages for the deployment.

Features:

  • Showcases different products Fake offers.
  • Users can log in or register an account.
  • FAQ page for users with questions.
  • Contact page.
