Vindina: Cloud Storage

Vindina is a fake company that wants to create a cloud storage system for their consumers.

While making the OFFGame project, my team found out we can access websites hosted in XAMPP by being in the same network as the server.

This project does exactly that but for data storage.

Tools used:

  • Bootstrap for the design to make it look professional.
  • PHP for the backend to validate inputs from users.
  • PHP connected to SQL to store the data entered by the user.
  • XAMPP for the deployment.
  • Second device that acts like the server.
  • Local internet for anyone connected can access to it.

Features:

  • Local IP addresses are automatically stored. This is a positive for users who does not want to disconnect their sessions from the server.
  • Users can store passwords, bookmarks, and files.
  • Users can register and log in to retrieve their stuff.
